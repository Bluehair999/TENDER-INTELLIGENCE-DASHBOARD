Overview

본 대시보드는 엑셀 입찰 데이터를 지능적으로 시각화하고, 복잡한 마감 일정을 한눈에 관리할 수 있도록 설계된 올인원(All-in-One) 통합 솔루션입니다.

  • 단일 파일 구조: HTML 하나에 디자인, 기능, 데이터가 모두 내장되어 별도 서버가 필요 없습니다.
  • 지능형 매칭: 입력된 데이터를 바탕으로 마감일(D-Day)을 자동 계산하고 시각화합니다.
  • 공유 최적화: 단일 파일 형태로 카톡, 이메일, 클라우드를 통해 간편하게 공유할 수 있습니다.

Dashboard & KPIs

시스템 구동 즉시 대시보드 상단 상시 요약 지표가 노출됩니다.

실시간 상태 요약

전체 건수, 참여(O) 건수, 낙찰(Winner) 건수를 실시간으로 요약 및 필터링합니다.

지능형 차트

상태별 분포, 수집 통계, 국가별 분석 차트를 통해 입찰 전략을 수립할 수 있습니다.

Smart Calendar

입찰 관리의 핵심인 마감일(Deadline)을 입체적으로 관리합니다.

상시 미니 달력

우측 대시보드 영역 상단에 미니 달력이 상시 노출되어 오늘과 이번 달 일정을 놓치지 않습니다.

상세 달력 팝업

날짜별 사업 코드가 매핑되며, 마우스 오버 시 상세 정보를 중앙에서 확인할 수 있습니다.

🚨 마감 임박 경고 (D-7): 기간 내 사업은 빨간색으로 깜빡이며(Blinking) 강력한 주의를 환기합니다.
🌑 지난 일정 처리: 마감이 도래한 과거 사업은 자동으로 회색(Gray) 처리되어 현재 일정과 명확히 구분됩니다.

Project List & Detail

강력한 데이터 테두리 내에서 상세한 사업 정보를 관리합니다.

  • 인터랙티브 리사이징: 테이블 열 너비를 마우스 드래그로 자유롭게 조절하여 최적의 뷰를 구성할 수 있습니다.
  • 스마트 검색/필터: 국가, 발주처, 상태별 다중 조건 검색을 지원합니다.
  • 보고용 요약 복사: 상세 패널에서 'Copy for Report'를 클릭하면 보고서용 정제된 텍스트가 즉시 복사됩니다.
  • 변경 이력 추적: 데이터 갱신 시 변경된 정보가 있을 경우 이력을 즉시 표시합니다.

Data Management

데이터의 연속성과 안전한 관리를 위한 기능을 제공합니다.

  • 엑셀 업로드: 상단 버튼을 통해 새로운 입찰 리스트 엑셀 파일을 드래그하여 즉시 업데이트합니다.
  • JSON 백업/복원: 현재의 작업 상태를 JSON 파일로 내보내거나 불러와 데이터 손실을 방지합니다.
  • 임베디드 업데이트: 전문가용 `EMBEDDED_DB` 수정을 통해 초기 로드 데이터를 영구히 고정할 수 있습니다.

Mobile Optimization

외근 중에도 스마트폰으로 모든 기능을 동일하게 활용할 수 있습니다.

  • 반응형 그리드: 모바일 화면 크기에 맞춰 KPI와 레이아웃이 자동 최적화됩니다.
  • 풀스크린 상세뷰: 좁은 화면에서도 텍스트를 읽기 편하도록 상세 패널이 전체 화면으로 열립니다.
  • 터치 인터페이스: 손으로 넘기기 편한 탭 구조와 버튼 크기를 적용했습니다.
